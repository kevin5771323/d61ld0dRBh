# 项目简介

## 核心功能
- 基于 YOLOv8 的 X 光安检图像检测系统，支持 8 类物品检测（饮料瓶、玻璃瓶、刀具、打火机、金属杯、充电宝、雨伞、扳手）
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供自定义置信度和 IOU 阈值设置
- 支持选择显示标签和原图
- 支持选择不同的检测模型
- 实时显示目标位置、目标总数和置信度信息
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存功能

## 适用场景
- 安检设备辅助检测系统
- X 光图像自动分析处理
- 安全检查自动化应用
- 模型训练与研究

## 技术特点
- 使用 Python 3.9 开发，依赖 OpenCV、PyQt5、Ultralytics 和 Torch 1.9 等技术栈
- 提供 PyCharm+Anaconda 或 VSCode+Anaconda 两种开发环境选择
- 包含 4000 张标注数据集，涵盖 8 类安检常见物品
- 内置训练好的模型，配置成功环境可直接使用
- 提供完整的 UI 界面和图标资源
